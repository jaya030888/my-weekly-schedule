# Created Weekly Timetable Using the Table Element
Name: Jaya Patel

**Repository Description:**

This repository contains a simple schedule webpage designed using basic web technologies.
The page displays my weekly routine and includes an interactive click feature.

**Features & Technologies Used:**

HTML Table Layout — *used for creating the structured weekly schedule*

CSS Styling — *background image, alternating row colors, custom fonts, spacing*

JavaScript Event Listener — *detects clicks anywhere on the page*

Page Toggle Logic — *clicking switches between two schedule pages (index.html ↔ schedule2.html)*

**Functions Performed:**

Displays a neatly formatted timetable

Organizes study subjects and timings clearly

Changes the page on click using a JS event listener
